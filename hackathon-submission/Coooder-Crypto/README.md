## project
- 项目名称: Web Assistant
> 描述: 描述这个项目是做什么

 让任何网页都变成可交互的知识库！与当前页面进行智能对话，获取即时解答。

### ✨ 特色功能
- 🎯 **智能上下文理解**：自动捕获并理解当前网页的内容
- 💬 **自然对话**：通过 AI 支持，实现与页面内容的自然交谈
- 🎨 **精美界面**：简洁现代的界面设计，流畅的动画效果和 Markdown 支持
- 🌙 **API支持**：用户自己选择自己喜欢的 API 进行配置
